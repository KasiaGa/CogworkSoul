**Cogwork Soul**

This is a Hollow Knight Silksong fan game made for fun, because I am a huge Silksong fan!!

Rant is a cogwork construct made by the Citadel. He gained subconsciousness and is trying to find out what he's made of, how the cogwork factory works and if it's possible to break free.


6 locations to discover
