# Cogwork Soul

This is a **Hollow Knight Silksong fan game** made for fun, because I am a huge Silksong fan!!

You wake up in a cogwork world made by the Citadel, among the cables, blinking devices, and pipes. You wake up as _Rant_. Rant is a cogwork construct made by the Citadel. In unknown circumstances, he gained subconsciousness and is determined to find out what he's actually made of, how the cogwork factory works and if it's possible to _break free_.
